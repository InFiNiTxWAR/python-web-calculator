# 🐍 Python Web Calculator

Aplicação web simples feita em **Python + Flask** com interface em **HTML e CSS**.

O projeto demonstra como conectar Python com uma interface web para criar um pequeno sistema de cálculo.

---

# 🚀 Tecnologias utilizadas

- Python
- Flask
- HTML
- CSS

---

# 📂 Estrutura do projeto

```
projeto-python
│
├── app.py
│
├── templates
│   └── index.html
│
└── static
    └── style.css
```

---

# ⚙️ Como rodar o projeto

### 1️⃣ Instalar Python

Baixe em:  
https://python.org

---

### 2️⃣ Instalar Flask

No terminal:

```
pip install flask
```

---

### 3️⃣ Rodar o projeto

Dentro da pasta do projeto execute:

```
python app.py
```

ou

```
py app.py
```

---

# 💻 Funcionalidade

- Interface web simples
- Entrada de dois números
- Cálculo da soma
- Resultado exibido na tela

---

Projeto criado para **praticar programação em Python e desenvolvimento web com Flask**.
